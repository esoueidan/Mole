# Mole
## Description

Mole is an Augmented Reality (AR) mobile game developed in C# using Unity, inspired by the classic arcade game Whack-A-Mole.

The game places the player in a real-world environment using their phone’s camera. Moles appear around the player for a limited amount of time, and the goal is to score as many points as possible by hitting the correct moles before they disappear.

## Features

* Augmented Reality gameplay on mobile devices
* Multiple difficulty levels
* Time-limited competitive scoring system
* Good and bad moles with different effects on the score
* Simple and intuitive touch controls

## Gameplay

1. Choose a difficulty level.
2. Moles will appear around you in AR space.
3. Tap on the correct moles to gain points.
4. Avoid hitting the wrong moles, or your score will decrease.
5. Try to get the highest score before time runs out.

## Technologies Used

* Unity
* C#
* Augmented Reality (AR) for mobile devices
